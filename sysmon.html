Understanding Sysmon for Threat Hunting

Sysmon or System Monitoring is a Windows System Service that monitors and log system activities to the windows log event. It provide detailed information about process creation, network connections and changes to file creation time. It collects the events and by using SIEM tools, it analyzes them, which further helps in identifying malicious activities in the network.
